TCTX64_20210701

TCTX64_20210701 is intended for 64-bit PCs running Windows 10, but is 
compatible with Windows 7 and Windows XP.

To install:

1.  Make a working folder [path]\TCT.

2.  Copy in the files TCTX64_20210701.exe, TctInfo.pdf, Readme.txt and 
    ScreenSettings.txt, together with the subfolders USER and Graphviz. 

3.  Run TCTX64_20210701.exe in Windows. For summary descriptions of the TCT 
    procedures, press I from the Main Options page. Details can be found in 
    the monograph: W. Murray Wonham and Kai Cai,  "Supervisory Control of 
    Discrete-Event Systems", Springer 2019.  

4.  If desired, adjust the screen display by clicking on the button in the 
    upper left corner and resetting the display parameters as listed in the 
    file ScreenSettings.txt included with this package.

TCT is free software intended primarily for teaching purposes, and is under 
continuous development.  It carries no guarantee of correctness whatsoever.
